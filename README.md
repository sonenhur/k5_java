## Stack and Queue

### Stack:

##### 후입선출(LIFO: Last In First Out)

##### 스택은 정해진 방향으로만, top으로 정한 곳에서만 접근한다

##### push: 삽입연산, pop: 삭제연산

##### 사용 사례 : 웹 브라우저 뒤로가기, 실행취소(undo) 등

### Queue:

##### 선입선출(FIFO: First In First Out)

##### inque: rear에서 이루어지는 삽입연산

##### deque: front에서 이루어지는 삭제연산

##### 사용 사례 : 대기열 순에 따라 우선순위가 정해지는 서비스 업무 등
